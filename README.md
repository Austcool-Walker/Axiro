# Axiro

Axiro is a discord bot, designed to be weird and fun at the same time. 

## Features:

* Basic encryption, decryption, and hash.
* Typical anime stuff.
* An economy run on chickens.
* Music capabilities
* Access the news.
* Do what admins need to do (kick, ban, etc.)
